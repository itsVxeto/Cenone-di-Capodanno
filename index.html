Nome Tipo Uso Vincoli Note
ageA int I >0 età prima persona
ageB int I >0 età seconda persona
ageC int I >0 età terza persona
nChampagne int I >=0 numero di bicchieri di champagne
nVegan int I >=0 controlla il numero di persone vegane
prezzoChampagne float I >0 prezzo dello champagne
prezzoDiscoteca float W
k=100€		 / prezzo della discoteca
babysitting float W
k=10€		 / prezzo del babysitting
piro float W
k=10€		 / prezzo giochi pirotecnici
vegan float W
k=10€		 / prezzo per i vegani o altro
piroTF str W "Si"||"No" controlla se si vuole accedere ai giochi pirotecnici o meno
veganTF str W "Si"||"No" controlla se ci sono persone vegane tra le persone
champagneTF str W "Si"||"No" controlla se ci sono persone che voglio lo champagne (maggiorenni)
costoA float W / costo di ageA
costoB float W / costo di ageB
costoC float W / costo di ageC
total float O / totale risultato

Operazioni:

ageA >= 18 → costoA = costoA + prezzoDiscoteca
ageA >= 6 and ageA <= 17 → costoA = 20
costoA = 10 + babysitting piroTF == "Si" → costoA = costoA + piro

ageB >= 18 → costoB = costoB + prezzoDiscoteca
ageB >= 6 and ageB <= 17 → costoB = 20
costoB = 10 + babysitting piroTF == "Si" → costoB = costoB + piro
ageC >= 18 → costoC = costoC + prezzoDiscoteca
ageC >= 6 and ageC <= 17 → costoC = 20
costoC = 10 + babysitting piroTF == "Si" → costoC = costoC + piro
total = costoA + costoB + costoC ageA >= 18 || ageB >= 18 || ageC >= 18 → champagneTF == "Si" → total = total + (nChampagne * prezzoChampagne) veganTF == "Si" → total = total (vegan * nVegan)
